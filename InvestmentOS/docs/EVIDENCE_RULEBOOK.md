# Evidence Rulebook

## Purpose

Defines the deterministic rules for collecting, validating, prioritizing, and managing investment evidence within InvestmentOS.

The rulebook ensures every recommendation is based on trusted, complete, current, and verifiable evidence.

---

## 1. Evidence Principles

- Evidence before opinion
- Trusted sources only
- Latest available data
- Complete where possible
- Traceable
- Explainable
- Never fabricate evidence

## 4. Mandatory Evidence

Every Long-term Analysis requires:

- Annual Report
- Latest Quarterly Results
- Financial Statements
- Valuation Metrics
- Business Overview
- Risk Factors

Analysis should not proceed without mandatory evidence unless explicitly downgraded to WARNING.

---

## 5. Freshness Rules

Market Data

- Real-time or latest available

Financial Data

- Latest reported quarter

News

- Last 90 days

Technical Indicators

- Latest trading session

Macro Data

- Latest published values

## 8. Evidence Validation Rules

Every evidence item must contain:

- Source
- Timestamp
- Category
- Reliability
- Status

---

## 9. Missing Evidence Rules

Mandatory Evidence Missing

→ FAIL

Optional Evidence Missing

→ WARNING

Old Evidence

→ Reduce Confidence

Conflicting Evidence

→ Flag for Review

---

## 10. Duplicate Rules

When duplicate evidence exists:

- Keep latest
- Prefer higher reliability
- Record discarded sources
