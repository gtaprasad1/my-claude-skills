# InvestmentOS - Project Map

## Overview

InvestmentOS is an AI-powered Investment Decision Support System.

Its purpose is to improve investment decision quality by combining structured investment knowledge with AI-powered reasoning.

This repository contains the knowledge, skills, and documentation required to build and continuously improve InvestmentOS.
